# Chitragupta

A note taking app.

Built on React and Firebase.

# Pre-requisites

- NodeJS : Install from [here](https://nodejs.org/en/).

# Installation

- `npm i` : Install dependencies.

# Running

- `npm run dev` : Build and launch app on localhost with Firebase emulators.
